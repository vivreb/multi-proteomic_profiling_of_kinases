# Multi-proteomic Profiling of Kinases

## What code is available?
Here I provide the code and any additional data needed to generate both main and supplementary figures presented in (publication link to follow soon). You should be able to reproduce any figure with the provided code and data. 

## How to use
All raw data is available on PRIDE (link to follow soon). All code and additional necessary data is available for download on github. Some files are very large and are therefore zipped. Please ensure that all files are unzipped and in the same folders before running the code.
